## Jak si to zpustit doma  

### Clonování stránky  

~~~bash  
git clone https://github.com/OndraValek/dotaznicek-node.git
~~~

### Dostaňte se do adresáře   

~~~bash  
cd dotaznicek-node
~~~

### Instalace npm  

~~~bash  
npm install
~~~

### Zapnutí serveru  

~~~bash  
node .
~~~
